# SportsBallFantasy — Image Generation Prompts

**Style Guide:** Punk-magic / Thrasher-Wizard aesthetic
- **Colors:** Electric Yellow #DFFF00, Deep Purple #BF00FF, Void Black #0A0A0A, Hot Magenta #FF006E, Acid Green #39FF14
- **Keywords:** punk rock, thrasher magazine, wizard grimoire, halftone texture, ink splatter, high contrast, dark fantasy, athletic wizards, competitive faeries
- **Vibe:** Aggressive, chaotic, beautiful. Never soft, never corporate.

---

## How to Use These Prompts

In Cursor, use the `/generate-image` skill:

```
/generate-image "PROMPT_HERE" WIDTH HEIGHT
```

All images save to `generated_art/` folder.

## Style Consistency Tips

When modifying prompts:

1. **Always include:** "punk rock aesthetic, thrasher magazine style, halftone texture"
2. **Color palette:** Stick to #DFFF00 (yellow), #BF00FF (purple), #0A0A0A (black)
3. **Mood:** "aggressive", "high contrast", "dark fantasy", "competitive"
4. **Avoid:** "cute", "soft", "pastel", "corporate", "minimalist"

---

## Batch Generation Strategy

For consistent style across multiple images:

1. Start with **exploration mode** for 3-5 variations of The Warden
2. Pick the best version
3. Use similar keywords for all other assets
4. Save favorites as reference images
5. Create `.image-style.json` if doing bulk generation (see generate-image skill docs)

---

## File Naming Convention

Generated images save to `generated_art/` with auto-generated names. For production use, rename to:
